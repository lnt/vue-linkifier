# v-linkify

> A simple Vue directive to turn URL's and emails into clickable links.
> NO dependencies!

## Installation

```js
npm install v-linkify
```

## Install globally

_main.js_
```js
import Vue from 'vue';
import vLinkify from 'v-linkify';

Vue.use(vLinkify)
```

## Install locally
_component.vue_
```js
import {vLinkify} from "v-linkify";

const component = {
   directives: {
      vLinkify
   }
}
```
